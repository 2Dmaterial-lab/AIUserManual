    通用提示词模板集合¶

    本文档收集了各场景下可直接复制使用的 AI 提示词模板。使用时将 [方括号] 中的内容替换为你的实际情况即可。

    本页导航：推理与深度分析 · 代码生成 · 报错调试 · 数据分析 · 文献与写作 · 工具学习 · 跨工具协作 · 按工具索引

    一、推理与深度分析类¶

    模板0：复杂问题深度推理¶

    我有一个复杂的问题需要深度分析，请开启推理/思考模式逐步解决：

问题描述：
[问题的完整上下文和背景]

涉及领域：
- [领域1]：[相关理论和公式]
- [领域2]：[相关理论和公式]

具体要求：
1. 逐步推导，不要跳步
2. 每一步解释你的推理依据
3. 如果遇到多种可能路径，列举并比较
4. 最终给出结论及置信度评估

参考数据：
[贴出相关数据、公式或代码]

    模板0B：让Agent自主完成多步任务¶

    我需要你作为 Agent 自主完成以下任务，过程中不需要我逐步确认：

任务目标：
[一句话描述最终目标，如：在项目中找到所有内存泄漏并修复]

项目路径：[项目根目录]
涉及文件：[文件范围或让 Agent 自行搜索]

约束条件：
- [不要修改XX文件/保持向后兼容/保持代码风格一致]

最终期望产出：
- [修复后的代码 / 一份分析报告 / 一个新的脚本]

请在完成后报告：做了什么、为什么这样做、需要我检查的地方。

    二、代码生成类¶

    模板1：生成数据处理脚本¶

    请帮我写一个 [语言] 脚本，完成以下数据处理任务：

数据描述：
- 文件格式：[CSV/Excel/TXT]
- 文件位置：[路径]
- 数据结构：[列名和含义，如：第一列是波长(nm)，第二列是透过率(%)]

处理要求：
1. [步骤1描述]
2. [步骤2描述]
3. [步骤3描述]

输出要求：
- 输出格式：[CSV/图表/屏幕打印]
- 如果输出图表，要求：[字体、线宽、轴标签等]

特殊情况：
- 需要处理的异常值：[描述]
- 空值处理方式：[跳过/填充/报错]

请添加详细注释，方便我理解和修改。

    模板2：生成绘图代码¶

    请帮我写 [MATLAB/Python] 绘图代码，制作一张 publication 级别的图表：

图表类型：[折线图/散点图/柱状图/热图/...]
数据描述：[数据来源和格式]

格式要求（论文发表标准）：
- 图片尺寸：[8cm x 6cm / 全栏宽度等]
- 字体：Helvetica，[8/10/12]pt
- 线宽：数据线 1.5pt，坐标轴 0.5pt
- 刻度方向：朝内
- 颜色方案：[黑白/彩色/色盲友好]
- 图例位置：[右上/下方/无图例]
- 输出格式：[EPS/PNG/PDF]，分辨率 [300dpi]

X轴：[标签和单位]
Y轴：[标签和单位]

    模板3：生成自动化脚本¶

    请帮我写一个 [语言] 脚本，实现以下自动化流程：

目标：[一句话描述自动化目标]

步骤：
1. [读取/下载/获取什么数据]
2. [对数据做什么处理]
3. [输出什么结果]

文件结构：
- 输入目录：[路径]
- 输出目录：[路径]
- 文件命名规则：[描述]

额外要求：
- 添加进度提示
- 添加错误处理（跳过出错文件继续执行）
- 添加日志记录
- 支持 [命令行参数/配置文件] 设置

    四、数据分析类¶

    模板6：实验数据分析方案¶

    我有一组实验数据，需要设计分析方案：

数据描述：
- 数据来源：[仪器/仿真/文献]
- 数据量：[行数/文件数]
- 数据格式：[列名和含义]
- 数据范围：[x轴范围、y轴范围等]

研究目标：
- [核心问题1]
- [核心问题2]

我目前考虑的分析方法：[描述你的想法]

请帮我：
1. 推荐最合适的分析方法
2. 列出分析步骤
3. 提供实现代码框架（[MATLAB/Python]）
4. 说明如何判断结果的可靠性

    模板6B：可复现数据分析检查¶

    请帮我检查下面的数据分析流程是否可复现、是否有统计或单位风险。

研究目标：
[说明要回答的科学问题]

数据来源：
- 文件/仪器/仿真：[说明]
- 样本量：[N]
- 单位：[列名与单位]
- 是否有重复实验：[有/无，如何重复]

当前分析步骤：
1. [步骤1]
2. [步骤2]
3. [步骤3]

当前代码或伪代码：
```[语言]
[贴出代码]
```

请输出：
1. 数据清洗、异常值处理、单位换算是否合理
2. 是否需要归一化、加权、误差棒或置信区间
3. 结果是否需要和空白组、对照组、文献值或仿真结果交叉验证
4. 最小可复现记录清单：软件版本、依赖包、随机种子、输入输出文件、图表生成命令
5. 可能影响论文结论的风险点

    模板7：曲线拟合¶

    我需要对实验数据进行曲线拟合：

数据描述：
- 自变量：[名称和单位]
- 因变量：[名称和单位]
- 数据点数：[N个]
- 数据特征：[是否有噪声/是否需要加权]

物理模型：
[给出数学公式，如 y = A*exp(-x/tau) + B]

或者：我不确定用什么模型，数据呈现 [描述趋势，如"指数衰减+振荡"]的特征。

拟合要求：
- 使用 [MATLAB/Python/Origin]
- 输出拟合参数及其标准差
- 输出 R-squared 和残差图
- 如果有多种候选模型，帮我比较哪个更合适

参数初始猜测（如果已知）：
- [参数名] ≈ [估计值]，因为 [物理原因]

    五、文献与写作类¶

    模板8A：文献初筛与证据表¶

    请帮我整理下面这批文献，目标是形成可核对的证据表。不要编造没有给出的信息；缺失项请写“需查原文”。

研究问题：
[一句话说明你要回答的问题]

候选文献列表：
[粘贴从 Web of Science / Google Scholar / CNKI / 期刊官网导出的标题、作者、年份、DOI、摘要]

请按表格输出：
- 论文题目
- 年份 / 期刊 / DOI
- 材料体系或研究对象
- 方法：实验 / 仿真 / 理论 / 综述
- 关键结论
- 与我的研究问题的关系
- 需要回到原文核对的内容
- 可信度初评：高 / 中 / 低，并说明原因

最后请给出：
1. 还缺哪些关键词或检索方向
2. 哪些论文必须精读
3. 哪些结论不能直接引用，必须查原文或补实验验证

    模板8：论文方法部分¶

    请帮我润色以下论文方法部分的文字，要求：
- 学术英语风格
- 过去时态
- 简洁精确，避免冗余表达
- 保持技术准确性
- 适当使用被动语态

原文：
[贴出你的草稿]

背景信息：
- 这篇论文的领域是：[领域]
- 目标期刊：[期刊名]
- 段落在整个方法部分的位置：[第几段/描述什么]

请给出修改后的版本，并说明主要修改了哪些地方及原因。

    使用提示¶

    1. 不要照搬：这些模板是起点，根据你的具体情况调整
    2. 越具体越好：填入的细节越多，AI 的回答质量越高
    3. 迭代优化：第一轮回答不完美很正常，追问和修正
    4. 保存好用的模板：当你发现某个模板特别有效，保存下来反复使用
    5. 逐步添加细节：如果不确定某个参数，先留空让 AI 问你
    6. 复杂问题用推理模式：涉及多步推导、公式变换、疑难 bug 时，使用模板 0 开启深度推理
    7. 多文件任务用Agent：代码重构、跨文件搜索替换，使用模板 0B 让 AI 自主闭环执行
    8. 文献引用要回源：模板 8A 只用于初筛和整理，正式引用必须回到 DOI、期刊官网或数据库核对
    9. 保留 AI 使用记录：涉及论文、代码或图表时，用模板 9B 留下可审计记录
