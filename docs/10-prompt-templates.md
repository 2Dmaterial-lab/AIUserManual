# 通用提示词模板集合

本文档收集了各场景下可直接复制使用的 AI 提示词模板。使用时将 `[方括号]` 中的内容替换为你的实际情况即可。

---

## 一、代码生成类

### 模板1：生成数据处理脚本

````
请帮我写一个 [语言] 脚本，完成以下数据处理任务：

数据描述：
- 文件格式：[CSV/Excel/TXT]
- 文件位置：[路径]
- 数据结构：[列名和含义，如：第一列是波长(nm)，第二列是透过率(%)]

处理要求：
1. [步骤1描述]
2. [步骤2描述]
3. [步骤3描述]

输出要求：
- 输出格式：[CSV/图表/屏幕打印]
- 如果输出图表，要求：[字体、线宽、轴标签等]

特殊情况：
- 需要处理的异常值：[描述]
- 空值处理方式：[跳过/填充/报错]

请添加详细注释，方便我理解和修改。
````

### 模板2：生成绘图代码

````
请帮我写 [MATLAB/Python] 绘图代码，制作一张 publication 级别的图表：

图表类型：[折线图/散点图/柱状图/热图/...]
数据描述：[数据来源和格式]

格式要求（论文发表标准）：
- 图片尺寸：[8cm x 6cm / 全栏宽度等]
- 字体：Helvetica，[8/10/12]pt
- 线宽：数据线 1.5pt，坐标轴 0.5pt
- 刻度方向：朝内
- 颜色方案：[黑白/彩色/色盲友好]
- 图例位置：[右上/下方/无图例]
- 输出格式：[EPS/PNG/PDF]，分辨率 [300dpi]

X轴：[标签和单位]
Y轴：[标签和单位]
````

### 模板3：生成自动化脚本

````
请帮我写一个 [语言] 脚本，实现以下自动化流程：

目标：[一句话描述自动化目标]

步骤：
1. [读取/下载/获取什么数据]
2. [对数据做什么处理]
3. [输出什么结果]

文件结构：
- 输入目录：[路径]
- 输出目录：[路径]
- 文件命名规则：[描述]

额外要求：
- 添加进度提示
- 添加错误处理（跳过出错文件继续执行）
- 添加日志记录
- 支持 [命令行参数/配置文件] 设置
````

## 使用提示

1. **不要照搬**：这些模板是起点，根据你的具体情况调整
2. **越具体越好**：填入的细节越多，AI 的回答质量越高
3. **迭代优化**：第一轮回答不完美很正常，追问和修正
4. **保存好用的模板**：当你发现某个模板特别有效，保存下来反复使用
5. **逐步添加细节**：如果不确定某个参数，先留空让 AI 问你
